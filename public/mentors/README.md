# Mentors photos

Сюда бросаем фото менторов для страницы `/mentors`.

Ожидаемые имена файлов:

- `1.jpg` — первый ментор
- `2.jpg` — второй ментор
- `3.jpg` — третий ментор
- `4.jpg` — четвёртый ментор

Если файл отсутствует — в кружке покажутся инициалы из имени (см. `src/pages/Mentors.jsx`).

Поменять путь/имя можно в `src/i18n/translations.js` в секции `mentors.list[*].photo`.
Квадратные фото работают лучше всего — они обрезаются под `border-radius: 9999` (полный круг).
